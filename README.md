Sistema Web Académico Accesible

Descripción

Proyecto de diseño de un sistema web académico tipo campus virtual, enfocado en accesibilidad y cumplimiento de la normativa WCAG 2.2 nivel AA.

El sistema permite la interacción entre estudiantes, profesores y administradores garantizando igualdad de acceso.

---

Objetivo

Diseñar una interfaz accesible que cumpla con criterios de usabilidad, inclusión y accesibilidad universal.

---

Roles de Usuario

- Estudiante con discapacidad visual parcial
- Profesor con limitación motora
- Estudiante con dislexia
- Administrador con conexión lenta

---

 Historias de Usuario

Se desarrollaron 8 historias de usuario alineadas con WCAG 2.2 AA:

- Navegación por teclado
- Contraste mínimo 4.5:1
- Skip links
- Formularios accesibles
- Tipografía adaptable
- Indicadores de estado
- Uso de landmarks ARIA
- Breadcrumbs accesibles

(Ver documentación en carpeta /documentos)

---

Criterios WCAG Aplicados

- 2.1.1 Keyboard Accessible
- 1.4.3 Contrast (Minimum)
- 2.4.1 Bypass Blocks
- 3.3.1 Error Identification
- 1.4.4 Resize Text
