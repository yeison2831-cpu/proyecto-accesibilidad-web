 DECISIONES DE DISEÑO

1. Contraste de colores
Se decidió utilizar una relación de contraste mínima de 4.5:1 entre texto y fondo.
Principio aplicado: Accesibilidad y usabilidad.
Normativa: WCAG 2.2 AA - 1.4.3 Contrast (Minimum).

2. Navegación por teclado
Todos los elementos interactivos serán accesibles mediante teclado (TAB).
Principio aplicado: Inclusión y accesibilidad universal.
Normativa: WCAG 2.2 AA - 2.1.1 Keyboard Accessible.

3. Uso de Skip Links
Se implementa un enlace para saltar directamente al contenido principal.
Normativa: WCAG 2.2 AA - 2.4.1 Bypass Blocks.

4. Formularios accesibles
Los campos tendrán etiquetas claras y mensajes de error descriptivos.
Normativa: WCAG 2.2 AA - 3.3.1 Error Identification.

5. Tipografía legible
Se utiliza tamaño mínimo 16px y estructura clara.
Normativa: WCAG 2.2 AA - 1.4.4 Resize Text.
